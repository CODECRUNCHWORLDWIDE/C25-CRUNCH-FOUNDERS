# Exercise 3 — Term-Sheet Read-Through

**Goal:** Read a real pre-seed/seed term sheet the way a founder should — not to lawyer every clause, but to find the four or five lines that actually decide your outcome, flag the boilerplate as boilerplate, and identify at least one term you'd push back on and why. You leave able to *brief a lawyer*, not replace one.

**Estimated time:** 45 minutes.

**Format:** This is an **annotation template**, not code. Pull a real sample term sheet (links below), read it, and fill in the tables here. Copy this file into your Week 8 repo and complete the blanks.

> **Education, not investment, tax, or legal advice.** A term sheet is a document you negotiate and sign with a lawyer. This exercise builds your judgment about *which terms to ask your lawyer about*.

---

## Step 0 — Get a real term sheet to read

Use one of these free, standard documents (all in resources.md too):

- **Y Combinator — Series Seed / post-money SAFE + side letters:** <https://www.ycombinator.com/documents/>
- **NVCA model term sheet:** <https://nvca.org/model-legal-documents/>
- **Cooley GO term-sheet generator and explainer:** <https://www.cooleygo.com/>

Read the whole thing once, slowly, without trying to understand every word. Then come back and annotate.

## Step 1 — Classify every section

Go section by section. For each, mark it **MATTERS** (it changes your ownership, control, or downside) or **BOILERPLATE** (standard, the same across deals, low-stakes). Don't overthink the boilerplate — the skill is *not* drowning in it.

| Section | MATTERS / BOILERPLATE | One-line note |
|---------|----------------------|---------------|
| Valuation / cap | | |
| Amount raised | | |
| Option pool (size & timing) | | |
| Pro-rata rights | | |
| Board composition | | |
| Liquidation preference | | |
| Voting / protective provisions | | |
| Anti-dilution | | |
| Confidentiality | | |
| Governing law | | |
| No-shop / exclusivity | | |
| Reps & warranties | | |

You should end up with roughly **4–6 MATTERS** rows and the rest BOILERPLATE. If you marked everything "MATTERS," you're reading like a lawyer, not a founder — re-read and be ruthless about what truly moves your outcome.

---

## Step 2 — The terms that matter, in your own words

For each term you marked MATTERS, write one plain-English sentence on what it does *to you*, and the number/value in this specific sheet:

```
Valuation / cap:        ____________________________________________
                        Value in this sheet: $__________

Option pool:            ____________________________________________
                        Size: ____%   ·   Timing: pre-money / post-money (circle one)

Pro-rata rights:        ____________________________________________
                        Granted? yes / no

Board composition:      ____________________________________________
                        Founder seats: ___  Investor seats: ___  Independent: ___

Liquidation preference: ____________________________________________
                        Multiple: ___x   Participating? yes / no
```

> **Founder-friendly defaults to recognize:** a **1x non-participating** liquidation preference, a board you still control (or no board change at pre-seed on SAFEs), an option pool you negotiated to the size you actually need, and pro-rata rights that are standard but bounded. Anything markedly worse than these is worth a conversation with counsel.

---

## Step 3 — Pick one term to negotiate

Choose **one** term you would push back on. State it, your ask, and your reasoning. Reason from your Week 1 control-vs-wealth lean and your Week 6 economics — not from a sense that "pushing back is what tough founders do."

```
Term I'd negotiate:     ____________________________________________

My ask:                 ____________________________________________

Why (tie to my lean / economics):
________________________________________________________________
________________________________________________________________
```

A strong example: *"The option pool is set at 15% pre-money. My hiring plan for the next 18 months needs maybe 8–10%. I'd ask to size it at 10%, because every point of an oversized pre-money pool comes straight out of my ownership before the investor's money even lands. If we hire faster than planned, we top it up at the next round."* That's a founder reasoning from arithmetic, not bravado.

## Stretch

- **Two sheets, side by side.** Read both a YC and an NVCA document and note where they differ. The YC SAFE is deliberately minimal; the NVCA priced-round docs are thorough. Note which terms only appear once you're doing a *priced* round.
- **The board math.** If a sheet adds an investor board seat, sketch what board control looks like after a *second* round. Founders often lose board control before they lose 50% ownership — find where that line is in your projection.
- **Red-team a "founder-friendly" claim.** Find one clause that's marketed as founder-friendly and argue, in two sentences, the case where it isn't. (Pro-rata, MFN, and "standard" pools are good candidates.)

## Hints

<details>
<summary>How do I tell boilerplate from substance fast?</summary>

Ask of each clause: *if this term were different, would my ownership, my control, or my downside in a bad outcome change?* If yes — it matters. If it's about process, confidentiality, jurisdiction, or standard legal hygiene that's identical across every deal — it's boilerplate. Governing law and confidentiality are almost always boilerplate; valuation, pool, board, and liquidation preference almost always matter.

</details>

<details>
<summary>I'm at pre-seed on SAFEs — there's barely a "term sheet"</summary>

Right — a SAFE *is* the short version, and that's part of the lesson. Annotate the SAFE itself: the cap, the discount, the MFN, the pro-rata side letter. Then read an NVCA priced-round sheet so you recognize the terms that *will* appear at your seed or Series A, before they appear under time pressure.

</details>
