## Glossary

| Term | Plain English |
|------|---------------|
| **MVP** | Smallest experiment that tests the riskiest assumption |
| **Riskiest assumption** | The most uncertain belief that, if false, sinks the idea |
| **Fake door / smoke test** | A page measuring interest before the thing exists |
| **Concierge** | Delivering the service by hand to a few customers |
| **Wizard of Oz** | Manual work hidden behind an automated-looking front |
| **Build-measure-learn** | The lean loop; minimize total time through it |
| **Validated learning** | A result that would actually change your mind |
| **Pivot** | A learned change of direction |
